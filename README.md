# 📊 Análisis de Inflación en Argentina

Análisis de la evolución del IPC (Índice de Precios al Consumidor) 
en Argentina usando datos oficiales del BCRA.

## Herramientas
- Python · Pandas · NumPy · Matplotlib
- Fuente: API pública del BCRA

## Contenido
- Evolución mensual del IPC con promedio móvil
- Inflación acumulada anual por período
- Detección de picos inflacionarios
- Tabla resumen estadístico por año
